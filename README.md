# VirgoSoft - Trade Forge

![Trade Forge Banner](banner.jpg)

A full-stack limit-order exchange mini engine built with Laravel and Vue.js, featuring real-time order matching, balance management with race condition safety, and commission-based trading for cryptocurrency pairs.
